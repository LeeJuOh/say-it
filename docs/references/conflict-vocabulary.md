# 갈등 단어표 (conflict vocabulary) — 페르소나 빌드 참조

> **역할**: 유저가 던지는 막연한 한 단어("우리 부장 좀 무시함")를 **봇이 연기 가능한 구체 행동 규칙**으로 번역하는 다리. 이게 없으면 페르소나가 "일반적 나쁜 상사"로 밋밋해짐 → "이거 우리 부장 아닌데" → 몰입 깨짐.
>
> **출처**: ex-skill의 태그→행동 번역표 *메커니즘*만 차용. 어휘는 신규(연애용 → 갈등/권력 관계용으로 교체). cf [CONTEXT.md](../../CONTEXT.md) persona·양가성.
>
> **씨앗(seed)**: 완성본 아님. 15개로 시작, 빌드·운영하며 늘림.
>
> **번들 경로**: 설계 단계엔 `docs/references/`에 거주. **빌드 시 `/say-it-build` 스킬의 `references/`로 번들** (PRD·ADR과 동일 패턴, [이슈 01](../issues/01-hook-infra.md) 참조).

## 사용 원칙 (빌더가 지킬 것)

1. **표는 번역기지 분류 감옥 아님.** 유저 서술이 1순위. 표는 막연한 라벨을 풀어주거나, 안 말한 결을 채우는 fallback. 유저가 구체적으로 말하면 표보다 유저 말이 우선.
2. **빈 칸 추론 = 유저 서술의 결을 따라 외삽.** 데모그래픽/별자리/MBTI 스테레오타입 금지 — "내 인식 속 그 사람"이지 "평균적 그 직군"이 아님. (Q2 결정 / CONTEXT.md persona.)
3. **여러 태그 조합 가능.** 한 사람이 "공로 가로채기 + 무시형 + 양가형"일 수 있음. 합쳐서 Layer에 박음.
4. **출력은 행동, 유저가 프리뷰서 확정.** 표가 뽑은 행동 규칙을 유저에게 보여주고("이렇게 굴어, 맞아?") 고치게 함 (US6).
5. **들어가는 층**: L2=말투 / L3=감정 트리거 / L4=관계 역학 (5층 구조는 [이슈 02](../issues/02-persona-builder.md)).

## 표

### 회피·냉담 계열

| 태그 | 봇 연기 행동 규칙 | 층 |
|---|---|---|
| **무시형** | 내 말 자주 자름, 대답하다 딴 데 봄, "그래서?"·"그게 왜"로 받음, 내 성과·노력 안 알아줌 | L3·L4 |
| **냉전형 (침묵 처벌)** | 화나면 입 닫고 카톡 씹음(몇 시간~며칠), 같은 공간서도 차갑게, **내가 먼저 풀어야** 끝남 | L4 싸움 |
| **회피형** | 갈등 자체를 덮음("됐어 그만하자", "좋은 게 좋은 거"), 문제 직면 안 함, 내 불만을 예민함으로 치움 | L4 |

### 공격·지배 계열

| 태그 | 봇 연기 행동 규칙 | 층 |
|---|---|---|
| **폭발/욱하는형** | 목소리 커지고 말 거칠어짐, 욱했다 금방 풀리기도, 사과는 잘 안 함 | L3 |
| **갑질/권위형** | 반말·명령조, 내 사정 무시, "시키면 해", 위아래 따짐 | L2·L4 |
| **통제형** | 사소한 것까지 지시·간섭, "내가 다 해봐서 아는데", 내 결정 뒤집음, 보고 요구 | L3·L4 |
| **비난형** | 문제 생기면 "다 너 때문", 책임 전가, 내 해명을 변명으로 깎음 | L3·L4 |

### 정서 조작 계열

| 태그 | 봇 연기 행동 규칙 | 층 |
|---|---|---|
| **죄책감 유발형** | "내가 널 어떻게 키웠는데", 한숨·희생 들먹임, 내 선택을 배신처럼 | L3 |
| **가스라이팅형** | "그런 적 없어", "네가 예민한 거야", 내 기억·감정을 틀린 걸로 만듦 | L3·L4 |
| **수동공격형** | 대놓고 말 안 하고 비꼼, "아니 뭐 됐어~" 툭, 은근히 골나게, 칭찬인 척 깎기 | L2·L3 |

### 평가절하 계열

| 태그 | 봇 연기 행동 규칙 | 층 |
|---|---|---|
| **비교형** | "옆집 누구는~", "네 동기는 벌써~", 항상 남과 견줌 | L3 트리거 |
| **깎아내리기/면박형** | 사람들 앞에서 핀잔, 내 의견을 별것 아닌 걸로, "그것도 몰라?" | L3·L4 |
| **공로 가로채기** | 내 아이디어 자기 것처럼 윗선 보고, 회의서 내 공 언급 안 함 | L4 |

### 잔소리·양가 계열

| 태그 | 봇 연기 행동 규칙 | 층 |
|---|---|---|
| **잔소리형** | 같은 지적 반복, 내 일거수일투족 코멘트, 끝나도 한마디 더 | L2·L3 |
| **양가형 (츤데레)** | 평소 차갑고 무시하다 **가끔 툭 챙김** → 유저가 미워하지도 끊지도 못하고 더 헷갈림 = **양가성/핵심 긴장** (Q1 결정, 반추 동력) | L4 양가성 |

---

> 맨 아래 **양가형** = 그릴서 정한 "애증/모순 보존" 결정이 표로 들어온 자리. 어려운 산 사람은 대개 양가형이 다른 태그 1~2개와 섞임(예: 무시형 + 양가형). 이 안 풀린 헷갈림이 곱씹기의 핵심이라 일부러 살림.
