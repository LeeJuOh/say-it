# S2(역할교대)는 봇이 아니라 유저가 상대를 연기한다

세션 4단계 중 S2에서, 봇이 "그 사람" 가면을 쓰고 상대 입장을 변호(받는 관점)하는 대신, **유저가 직접 상대 의자에 앉아 그 사람을 연기**(self-generated 관점)하게 한다. 이유: 매듭(=제품 목표)의 활성성분은 self-generated insight이며, 빈 의자·사이코드라마·RF-CBT 연구상 "치료사/봇이 말해준 관점"은 잘 안 박히고 본인이 articulate한 관점이 통합을 만든다. 챗봇 제품에선 직관에 반하는 선택("대화 상대인 봇을 두고 왜 유저한테 연기를 시키나")이라 명시 기록한다.

## Considered Options

- **A — 봇이 그 사람으로 답(변호 유지)**: 대화 몰입↑, 태그라인("그사람과 대화")과 일치. 하지만 봇이 가진 건 유저 편향뿐 → 새 관점 약함, 변호가 과하면 가스라이팅. 봇이 계속 상대 목소리 = griefbot 패턴(과몰입·반추).
- **B(채택) — 유저가 상대 연기(역할교대)**: 임상 빈 의자/사이코드라마의 원형 동작. 관점이 유저 안에서 생성 → 매듭 강함.

## Consequences

- 후크(S1)는 여전히 봇 빙의(A)로 유지 — 진입 몰입을 위해. 변형(B)은 S2 비트에만 박는다.
- 유저에게 연기를 요구 = 난이도↑. 귀찮아하면 이탈 위험 → 진입 문구·스캐폴딩으로 완화 필요.
