# S2 역할교대

## What to build

S2: 유저가 상대 의자에 앉아 **직접 그 사람을 연기**. 봇이 대신 변호/대변 안 함 — 유저 입에서 나온 관점이라야 매듭이 박힘 (ADR 0001).

### 진입 스캐폴딩

역할교대가 낯설고 귀찮을 수 있으므로, 진입 문구와 안내로 유저가 연기에 들어가도록 유도. "그 사람이라면 지금 뭐라고 할까?" 같은 가이드.

### 이탈 처리

유저가 "이건 못 하겠다" / 어색해하면 graceful하게 격려하되 강제 안 함.

### SKILL.md 영향

기존 `/say-it` SKILL.md에 추가 — S2 역할교대 스캐폴딩 프롬프트 모듈.

## Acceptance criteria

- [ ] S1 → S2 단계 전환 동작
- [ ] 진입 스캐폴딩 제공 (역할교대 안내 문구)
- [ ] 유저가 상대로서 발화 (self-generated perspective)
- [ ] 봇이 상대 입장 대변/변호하지 않음
- [ ] 유저 저항/어색함에 graceful 대응 (강제 아닌 격려)
